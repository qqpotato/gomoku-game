## 游戏功能说明

### 主菜单
- 双人对战按钮
- 人机对战按钮
- 退出游戏按钮

### 游戏界面
- 15×15 标准棋盘
- 木纹色背景
- 黑白棋子（带高光效果）
- 最后落子位置高亮显示
- 当前玩家提示
- 游戏模式显示

### 游戏功能
- **双人对战**：本地同屏，黑白轮流落子
- **人机对战**：玩家执黑，AI 执白
- **悔棋**：双人对战可悔一步，人机对战可悔两步
- **重新开始**：清空棋盘，重新开始
- **返回菜单**：返回主菜单

### AI 特性
- 基于评分系统的智能 AI
- 评估各种棋型（五连、活四、冲四、活三等）
- 同时考虑进攻和防守
- 优先选择中心位置
- 能够识别并阻止玩家的获胜棋型

## AI 算法说明

AI 使用基于评分系统的评估函数：

1. **棋型评分**：
   - 五连：100,000 分
   - 活四：10,000 分
   - 冲四：1,000 分
   - 活三：1,000 分
   - 冲三：100 分
   - 活二：100 分
   - 冲二：10 分

2. **评估策略**：
   - 对每个空位计算进攻分数
   - 对每个空位计算防守分数（阻止对手）
   - 综合评分 = 进攻分数 + 防守分数 × 0.9
   - 添加位置加成（中心位置更优）

3. **开局策略**：
   - 第一步优先选择中心位置

## 打包说明

### App 打包
使用 py2app 将 Python 脚本打包成独立的 macOS 应用程序：
```bash
python setup.py py2app
```

生成的 App 包含：
- 完整的 Python 运行时
- Pygame 库及所有依赖
- SDL2 等系统库
- 自动代码签名

### DMG 打包
使用 create-dmg 和 hdiutil 创建安装镜像：
```bash
create-dmg --volname "五子棋游戏" ... Gomoku.dmg dist/五子棋游戏.app
```

DMG 特点：
- 压缩后大小：28MB
- 包含应用程序和 Applications 快捷方式
- 友好的安装界面
