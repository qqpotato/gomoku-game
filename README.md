# 五子棋游戏 (Gomoku Game)

一个使用 Python + Pygame 开发的完整五子棋游戏。

## 功能特性

- ✅ 双人对战模式（本地同屏）
- ✅ 人机对战模式（带智能 AI）
- ✅ 完整的 GUI 界面（棋盘、棋子、菜单、提示信息）
- ✅ 悔棋功能（双人对战模式）
- ✅ 重新开始
- ✅ 返回主菜单
- ✅ 退出游戏

## 系统要求

- Python 3.7+
- macOS / Windows / Linux
- Pygame 2.0+

## 安装依赖

```bash
pip install pygame
```

## 运行游戏

```bash
python gomoku.py
```

## 游戏规则

1. 黑棋先行，双方轮流落子
2. 先连成五子（横、竖、斜均可）者获胜
3. 棋盘为 15×15 路
4. 双人对战模式支持悔棋
5. 人机对战模式中，玩家执黑，AI 执白

## 操作说明

- **鼠标左键点击**：落子 / 选择菜单
- **悔棋**：点击"悔棋"按钮（仅双人对战模式）
- **重新开始**：点击"重新开始"按钮
- **返回菜单**：点击"返回菜单"按钮

## 打包说明

### macOS 打包

```bash
# 安装 py2app
pip install py2app

# 打包成 App
python setup.py py2app

# 生成 DMG（需要 create-dmg）
create-dmg \
  --volname "五子棋游戏" \
  --window-pos 200 120 \
  --window-size 600 400 \
  --icon-size 100 \
  --app-drop-link 400 200 \
  "五子棋游戏.dmg" \
  "dist/Gomoku.app"
```

### Windows 打包

**方法一：使用脚本（推荐）**

在 Windows 上双击运行：
- `build-windows.bat`（命令提示符）
- 或右键 `build-windows.ps1` → "使用 PowerShell 运行"

生成的 exe 文件位置：`dist\五子棋游戏.exe`

**方法二：手动打包**

```cmd
# 创建虚拟环境
python -m venv venv
venv\Scripts\activate

# 安装依赖
pip install pygame pyinstaller

# 打包成 exe
pyinstaller --onefile --windowed --name "五子棋游戏" gomoku.py
```

**方法三：GitHub Actions 自动构建**

推送代码到 GitHub 后，自动构建 Windows exe（见 `.github/workflows/build-windows.yml`）

### Linux 打包

```bash
# 安装依赖
pip install pygame pyinstaller

# 打包
pyinstaller --onefile --windowed --name "gomoku" gomoku.py
```

## AI 说明

本游戏使用基于评分系统的简单 AI：
- 评估各种棋型（五连、活四、冲四、活三等）
- 同时考虑进攻和防守
- 优先选择中心位置
- 能够识别并阻止玩家的获胜棋型
